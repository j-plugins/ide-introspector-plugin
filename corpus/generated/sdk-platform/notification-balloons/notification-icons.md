Customize the notification icon according to [message severity](https://plugins.jetbrains.com/docs/intellij/balloon.html#message-severity).
Use the constants from [NotificationType](https://github.com/JetBrains/intellij-community/tree/idea/261.24374.151/platform/ide-core/src/com/intellij/notification/NotificationType.java).

1. Generic information: `INFORMATION`.

2. Disruptions or events requiring user action: `WARNING`.

3. Critical events or states: `ERROR`.

When possible, use a plugin or functionality icon instead of the Information icon.
See the [Icons](#icons) section for more information.
