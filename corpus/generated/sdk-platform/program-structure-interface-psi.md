# Program Structure Interface (PSI)

The Program Structure Interface, commonly referred to as PSI, is the layer in the IntelliJ Platform responsible for parsing files and creating the syntactic and semantic code model that powers so many of the platform's features.

* [PSI Files](https://plugins.jetbrains.com/docs/intellij/psi-files.html)

* [File View Providers](https://plugins.jetbrains.com/docs/intellij/file-view-providers.html)

* [PSI Elements](https://plugins.jetbrains.com/docs/intellij/psi-elements.html)

Tip:

See [useful tools](https://plugins.jetbrains.com/docs/intellij/explore-api.html#internalMode) on how to inspect the PSI structure and its properties.
